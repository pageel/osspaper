# OSS English-First Governance

> Agent governance rule for configuration language.

## Scope
- [x] Project `osspaper`

## Triggers
- Creating or modifying any file in `.agents/`, `.github/`, `scripts/`, or root configs.

## Rules
1. **MUST** write all internal agent configurations, rules, skills, specs, and workflows in English.
2. **MUST** use English for all code variables, commit messages, and PR descriptions.
3. **SHOULD** communicate with the local user in their preferred language (e.g., Vietnamese) but keep technical artifacts strictly in English to support global open-source contributors.
